## AI Lead Qualification System

### Problem

Businesses waste time calling unqualified leads.

### Solution

This automation automatically calls and filters leads using AI.

### Features

* Auto call leads
* Detect valid/invalid phone numbers
* Filter bad leads
* Save qualified leads to Google Sheets

### Tech Stack

* n8n
* OpenAI
* Google Sheets
* API integrations

### How it Works

1. Lead enters system (form or sheet)
2. System calls the lead
3. AI analyzes response
4. Qualified leads are saved

### Results

* Saves 3–5 hours per day
* Filters 60–80% bad leads

### Setup

1. Import JSON into n8n
2. Add your API keys
3. Run the workflow
